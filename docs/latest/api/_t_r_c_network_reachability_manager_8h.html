Network Reachability

The following constants are provided by TRCNetworkReachabilityManager as possible network reachability statuses.

enum { TRCNetworkReachabilityStatusUnknown, TRCNetworkReachabilityStatusNotReachable, TRCNetworkReachabilityStatusReachableViaWWAN, TRCNetworkReachabilityStatusReachableViaWiFi, }

TRCNetworkReachabilityStatusUnknown The baseURL host reachability is not known.

TRCNetworkReachabilityStatusNotReachable The baseURL host cannot be reached.

TRCNetworkReachabilityStatusReachableViaWWAN The baseURL host can be reached via a cellular connection, such as EDGE or GPRS.

TRCNetworkReachabilityStatusReachableViaWiFi The baseURL host can be reached via a Wi-Fi connection.

◆ TRCNetworkingReachabilityDidChangeNotification

NSString* const TRCNetworkingReachabilityDidChangeNotification

Posted when network reachability changes.

This notification assigns no notification object. The userInfo dictionary contains an NSNumber object under the TRCNetworkingReachabilityNotificationStatusItem key, representing the TRCNetworkReachabilityStatus value for the current network reachability.

Warning
In order for network reachability to be monitored, include the SystemConfiguration framework in the active target's "Link Binary With Library" build phase, and add #import <SystemConfiguration/SystemConfiguration.h> to the header prefix of the project (Prefix.pch).
